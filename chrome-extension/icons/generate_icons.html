# instructions
  1. Click download all and save the 3 files to chrome-extension/icons/
  2. Replace the existing icon16.png, icon48.png, icon128.png
  3. Reload the extension at chrome://extensions/

Or run node icons/generate.mjs from the project root if you have sharp installed.
